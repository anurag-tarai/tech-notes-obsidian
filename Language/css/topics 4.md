
### 1️⃣ CSS Basics

- Difference between **inline, internal, and external CSS**
    
- Syntax: `selector { property: value; }`
    
- Understanding **selectors**: element, class `.class`, id `#id`, attribute `[type="text"]`
    

### 2️⃣ Box Model

- Understand **margin, border, padding, content**
    
- Difference between `content-box` and `border-box`
    
- Why it matters in layout and spacing
    

### 3️⃣ Positioning & Display

- `display: block`, `inline`, `inline-block`, `none`
    
- `position: static`, `relative`, `absolute`, `fixed`, `sticky`
    
- Difference between **static vs relative vs absolute positioning**
    

### 4️⃣ Flexbox (Most Important)

- Basic **layout tool** in modern web development
    
- Properties: `display: flex`, `justify-content`, `align-items`, `flex-direction`, `flex-wrap`
    
- Important for **React component layout** instead of old float-based layouts
    

### 5️⃣ Units & Sizes

- Absolute units: `px`, `pt`
    
- Relative units: `%`, `em`, `rem`
    
- Why relative units matter for responsive design
    

### 6️⃣ Colors & Backgrounds

- `color`, `background-color`, `background-image`, `linear-gradient`
    
- Not needed in depth, just basics for buttons, highlights
    

### 7️⃣ CSS Selectors You Must Know

- Class `.class`, ID `#id`, element `p`, `div`
    
- Descendant `.parent .child` and child `>`, sibling `+`
    
- Pseudo-classes: `:hover`, `:focus`, `:first-child`
    

### 8️⃣ CSS Specificity

- Inline > ID > Class > Element
    
- Why some styles override others (helpful in React CSS issues)
    

### 9️⃣ Responsive Basics

- `@media` queries for screen size
    
- Using `%` or `rem` units for layout
    
- `max-width`, `min-width` usage
    

### 10️⃣ CSS in React

- Inline styles: `<div style={{ color: "red" }}>`
    
- Class names: `className="container"` instead of `class`
    
- CSS Modules / Styled Components (basic awareness)
