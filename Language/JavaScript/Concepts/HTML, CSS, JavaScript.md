### ✅ **HTML**
- Structure/content of the page
- Defines _what_ you see
- Example: headings, paragraphs, buttons

`<p>Hello</p>`

---
### ✅ **CSS**
- Styling/appearance
- Defines _how it looks_
- Example: colors, layout, fonts

`p {color: blue; }`

---
### ✅ **JavaScript**
- Behavior/interaction
- Defines _what it can do_
- Makes pages dynamic (change content, handle clicks, animations)

`document.querySelector("p").textContent = "Changed with JavaScript!";`

---
### ⭐ **Short difference in one line:**

**HTML = structure, CSS = style, JavaScript = behavior/logic.**
