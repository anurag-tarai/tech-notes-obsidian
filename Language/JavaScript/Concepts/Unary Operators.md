- **Definition:** Operators that work on **only one operand**.
    
- **Purpose:** Usually used for **type conversion, negation, or checking type**.
## Common Unary Operators

| Operator | Description                    | Example        | Output         |
| -------- | ------------------------------ | -------------- | -------------- |
| `+`      | Converts value to number       | `+"42"`        | `42`           |
| `-`      | Converts to number and negates | `-"42"`        | `-42`          |
| `!`      | Logical NOT (boolean negation) | `!true`        | `false`        |
| `typeof` | Returns type of value          | `typeof 42`    | `"number"`     |
| `void`   | Returns `undefined`            | `void 0`       | `undefined`    |
| `++`     | Increment by 1                 | `let a=1; a++` | `1` (then a=2) |
| `--`     | Decrement by 1                 | `let a=1; a--` | `1` (then a=0) |
