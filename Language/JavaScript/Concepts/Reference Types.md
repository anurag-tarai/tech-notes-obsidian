 Non-primitive = mutable + stored by reference.
### 8. **Object**
Includes object literals, arrays, functions, dates, etc.

#### Object
```js
let user = { name: "Anurag", age: 20 };
```
#### Array
```js
let nums = [1, 2, 3];
```
#### Function
```js
function add(a, b) {
  return a + b;
}
```

# Built-in Object Types

- **Object** → base object
- **Array** → indexed collection
- **Date** → date/time
- **RegExp** → regular expressions
- **Map** → key-value pairs
- **Set** → unique value
- **Error** → error handling
- **Promise** → async operations
- **Typed Arrays** →
    - Int8Array
    - Int16Array
    - Int32Array
    - Float16Array
    - Float32Array
    - Float64Array
    - BigInt64Array
_(Plus others like Math, WeakMap, WeakSet, etc.)_
