|Term|Meaning|Where it appears?|
|---|---|---|
|**Parameters**|Variables **defined** in the function|In the **function definition**|
|**Arguments**|Actual **values passed** to the function|In the **function call**|
```js
function add(x, y) {   // x and y = parameters
  return x + y;
}


add(5, 10);  // 5 and 10 = arguments
```
