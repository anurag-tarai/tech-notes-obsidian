The **DOM Tree** is a hierarchical, node-based structure built by the browser when parsing HTML.
- Js use it to access, modify content & listen to user interaction on webpage.

# Types of Nodes
1. **Document Node**
2. **Element Node** (div, p, h1)
3. **Text Node** ("Hello")
4. **Attribute Node** (id="", class="")
5. **Comment Node** (<!-- comment -->)

`Note` :
**Only Element Nodes and Text Nodes matter for 99% DOM work**.
