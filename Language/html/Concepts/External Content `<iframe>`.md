Used to embed **external pages or content** (YouTube, maps, other sites).

```html
<iframe 
  src="https://www.youtube.com/embed/xyz"
  title="YouTube video"
  allowfullscreen>
</iframe>
```

**Key attributes:**
- `src` → URL of embedded content
- `title` → Accessibility (important)
- `allowfullscreen` → Enables fullscreen mode
