A **tag** is the **markup keyword** enclosed in angle brackets `<>`. It tells the browser **what kind of element it is**.

#### Types of tags:

- **Opening tag** ‚Üí `<p>`
    
- **Closing tag** ‚Üí `</p>`
    
- **Self-closing / Empty tag** ‚Üí `<img />`, `<br />`, `<hr />`
    

#### Example:

```html
<p>
```

üëâ This is a **tag only**, not the element itself.

### 2Ô∏è‚É£ HTML Element

An **element** is the **complete structure**:

> Opening tag + content (optional) + closing tag

#### Examples:

**Normal element (with content):**

```html
<p>This is a paragraph</p>
```

- `<p>` ‚Üí opening tag
    
- `This is a paragraph` ‚Üí content
    
- `</p>` ‚Üí closing tag
    

**Self-closing / empty element (no content):**

```html
<img src="photo.jpg" alt="Profile photo" />
```

- `<img />` ‚Üí self-closing **element**
    
- `src` and `alt` ‚Üí attributes
    

> ‚úÖ Even without content, a self-closing tag is still a **complete HTML element**.
