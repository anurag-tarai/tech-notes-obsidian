These attributes can be applied to **most HTML elements**:

| Attribute         | Purpose                                |
| ----------------- | -------------------------------------- |
| `id`              | Unique identifier for the element      |
| `class`           | Group elements for styling or JS       |
| `title`           | Provides **tooltip text** on hover     |
| `style`           | Inline CSS styling                     |
| `hidden`          | Hides the element (`display: none`)    |
| `contenteditable` | Makes element **editable by the user** |
