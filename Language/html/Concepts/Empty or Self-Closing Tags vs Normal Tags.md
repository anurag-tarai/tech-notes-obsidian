### 1️⃣ Empty / Self-Closing Tags
- Do **not have any content** between opening and closing.
- Represented as a single tag, sometimes with a trailing slash.
- **Cannot have inner content**.
#### Examples:
```html
<img src="photo.jpg" alt="Photo" />
<br />
<hr />
<input type="text" />
```
---

### 2️⃣ Normal Tags

- Have **opening and closing tags**.
- Can contain **text, other inline, or block elements**.
- Form a complete **HTML element**.
#### Examples:

```html
<p>This is a paragraph</p>
<div>
  <h1>Heading</h1>
</div>
<a href="https://google.com">Link</a>
```
