| Attribute     | Purpose                                 |
| ------------- | --------------------------------------- |
| `name`        | Identifies input when sending to server |
| `id`          | Links input to `<label>` using `for`    |
| `value`       | Default value or sent value             |
| `placeholder` | Hint for the user                       |
| `required`    | Makes field mandatory                   |
| `maxlength`   | Max characters allowed                  |
| `pattern`     | Regex pattern to validate input         |
| `disabled`    | Makes input uneditable                  |
| `readonly`    | Shows value but cannot edit             |
