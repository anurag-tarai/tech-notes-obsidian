- The `<head>` tag contains **meta-information about the HTML document**.
- This information is **not displayed directly** on the web page.
- Common uses: defining the title, linking CSS, adding metadata, scripts, or fonts.
### Purpose of Common `<head>` Tags
|Tag|Purpose|Example|
|---|---|---|
|`<title>`|Sets the **title of the page** shown on the browser tab|`<title>My Page</title>`|
|`<meta>`|Provides **metadata** like character set, author, viewport for responsive design|`<meta charset="UTF-8">` `<meta name="viewport" content="width=device-width, initial-scale=1.0">`|
|`<link>`|Links **external resources** like CSS or icons|`<link rel="stylesheet" href="style.css">`|
|`<style>`|Adds **internal CSS** directly inside the HTML|`<style> h1 { color: red; } </style>`|
