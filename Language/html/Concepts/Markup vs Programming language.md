## Markup Language
A **markup language** is used to **structure and describe content**.
#### Key characteristics:
- ❌ No logic or decision making
- ❌ No variables, loops, or conditions
- ✅ Describes **what the content is**
- ✅ Helps browsers/tools understand content structure

#### Example: 
**HTML** : (HyperText Markup Language)
```html
<h1>Welcome</h1>
<p>This is a paragraph</p>
```
**XML** : (eXtensible Markup Language)
```xml
<user>
  <name>Anurag</name>
  <age>23</age>
</user>
```

## Programming Language
A **programming language** is used to **write logic and instructions** that a computer can execute.
#### Key characteristics:
- ✅ Has variables
- ✅ Supports conditions (if/else)
- ✅ Supports loops
- ✅ Can perform calculations and decisions

#### Example: **JavaScript**, Java, C, Python
```js
let age = 20;
if (age >= 18) {
  console.log("Adult");
}
```
