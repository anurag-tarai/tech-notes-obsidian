- `<ul>` (**Unordered List**)
    - Creates a list **without order** (bullets).
        
- `<ol>` (**Ordered List**)
    - Creates a list **with order** (numbers/letters).
        
- `<li>` (**List Item**)
    - Represents **each item** inside `<ul>` or `<ol>`.
        
```html
<ul>
  <li>Apple</li>
  <li>Banana</li>
</ul>

<ol>
  <li>Step One</li>
  <li>Step Two</li>
</ol>
```
