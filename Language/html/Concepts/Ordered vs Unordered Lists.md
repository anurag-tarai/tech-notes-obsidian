Use `<ol>` when order matters, `<ul>` when it doesn’t.

| Feature        | Ordered List (`<ol>`) | Unordered List (`<ul>`) |
| -------------- | --------------------- | ----------------------- |
| Order matters  | ✅ Yes                 | ❌ No                    |
| Default marker | Numbers (1, 2, 3)     | Bullets (•)             |
| Use case       | Steps, rankings       | Menus, features         |
