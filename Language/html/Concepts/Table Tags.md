### `<table>`
- Root element of the table
- Wraps the entire table

---
### `<tr>` (Table Row)
- Defines a **row** in the table
- Can contain `<th>` or `<td>`

---
### `<th>` (Table Header Cell)
- Represents **header cells**
- Text is **bold and centered by default**
- Used for column or row headings

```html
<th>Name</th>
```

---

### `<td>` (Table Data Cell)
- Represents **actual data**
- Normal text alignment

```html
<td>Anurag</td>
```

---
### `<caption>`
- Provides a **title or description** for the table
- Improves **accessibility and SEO**
- Placed **immediately after `<table>`**

```html
<table>
  <caption>User Details</caption>
</table>
```
