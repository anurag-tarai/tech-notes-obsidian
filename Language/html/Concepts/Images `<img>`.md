- `<img>` is a **self-closing tag** used to display images.
- `src` specifies the image path (relative or absolute).
- `alt` provides **alternative text** if the image fails to load.

```html
<img src="images/logo.png" alt="Company logo">
```

**Importance of `alt`:**
- Improves **accessibility** (screen readers).
- Helps **SEO**.
- Displays text if image is broken.
