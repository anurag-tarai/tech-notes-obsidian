HTML is flexible, XHTML is strict and follows XML rules.

| Feature           | HTML                                                           | XHTML                                                            |
| ----------------- | -------------------------------------------------------------- | ---------------------------------------------------------------- |
| Full Form         | HyperText Markup Language                                      | eXtensible HyperText Markup Language                             |
| Syntax Strictness | Flexible, allows missing closing tags, lowercase not mandatory | Very strict, all tags must be closed, lowercase, properly nested |
| Case Sensitivity  | Not case-sensitive (`<P>` same as `<p>`)                       | Case-sensitive (`<p>` only, `<P>` invalid)                       |
| Error Handling    | Browsers try to fix errors                                     | Browsers may not render if syntax is wrong                       |
| Compatibility     | Works on all browsers                                          | Must follow XML rules for strict parsing                         |
| Example           | `<br>`                                                         | `<br />`                                                         |
