
| Element                   | Purpose                   | Example                                                  |
| ------------------------- | ------------------------- | -------------------------------------------------------- |
| `<input>`                 | Single-line input         | `<input type="text" name="username">`                    |
| `<textarea>`              | Multi-line input          | `<textarea name="message"></textarea>`                   |
| `<select>`                | Dropdown list             | `<select name="country"><option>India</option></select>` |
| `<option>`                | Options inside `<select>` | See above                                                |
| `<button>`                | Submit or custom button   | `<button type="submit">Submit</button>`                  |
| `<label>`                 | Describes form fields     | `<label for="email">Email:</label>`                      |
| `<fieldset>`              | Groups related fields     | `<fieldset>...</fieldset>`                               |
| `<legend>`                | Caption for `<fieldset>`  | `<legend>Personal Info</legend>`                         |
| `<input type="radio">`    | Single choice among many  | `<input type="radio" name="gender" value="male">`        |
| `<input type="checkbox">` | Multiple choices          | `<input type="checkbox" name="hobby" value="reading">`   |
| `<input type="password">` | Hidden text input         | `<input type="password" name="pass">`                    |
| `<input type="email">`    | Validates email           | `<input type="email" name="email">`                      |
| `<input type="number">`   | Numeric input             | `<input type="number" name="age">`                       |
| `<input type="file">`     | File upload               | `<input type="file" name="resume">`                      |
| `<input type="hidden">`   | Hidden field              | `<input type="hidden" name="id" value="123">`            |
| `<input type="submit">`   | Sends form                | `<input type="submit" value="Send">`                     |
| `<input type="reset">`    | Clears form               | `<input type="reset" value="Reset">`                     |
