| Feature      | `id`                        | `class`                                 |
| ------------ | --------------------------- | --------------------------------------- |
| Uniqueness   | Must be **unique** per page | Can be **shared by multiple elements**  |
| Selector use | `#id` in CSS/JS             | `.class` in CSS/JS                      |
| Purpose      | Identify a single element   | Group elements for styling or scripting |
