## Definition
**Polymorphism** in Java means **“one interface, many forms.”**  
It allows a **single method or object reference** to exhibit **different behaviors** depending on the context.

---
## Real-Life Analogy

A person can play different roles:
- At home → **Father**
- At school → **Teacher**
- With parents → **Son**

Similarly, in Java, **the same method name** can perform **different actions**.
