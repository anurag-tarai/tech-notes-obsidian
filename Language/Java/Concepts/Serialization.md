**Serialization** is the process of **converting a Java object into a byte stream**.

 >The byte stream is **platform-independent**.
### Serialization is used when:
- You want to **persist object state**
- You want to **send objects over network**
- You want **caching**
- Used in **RMI, JPA, Hibernate, Spring**
### Real-life examples:
- Saving user session
- Sending objects between microservices
- Storing objects in files
