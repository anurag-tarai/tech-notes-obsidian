I chose Java because it strongly follows core programming principles and helps build a solid foundation in software development.

By learning Java, I strengthened my understanding of object-oriented programming, collections, multithreading, exception handling, and JVM concepts.

Java also has powerful frameworks like Spring and Hibernate that allow us to build scalable web and enterprise applications.

It is platform-independent, has strong community support, and is widely used by large organizations for backend and enterprise systems. That stability and demand made me choose Java as my primary language.
