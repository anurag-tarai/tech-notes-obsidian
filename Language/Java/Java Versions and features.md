
| Java Version | Release Year | Major Features / Changes                                                                                                                                   |
| ------------ | ------------ | ---------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Java 8**   | 2014         | Lambda expressions, Streams API, new Date/Time API (java.time), default methods in interfaces, Optional class.                                             |
| **Java 9**   | 2017         | Module system (Project Jigsaw), JShell REPL, improved Stream API, multi-release JARs.                                                                      |
| **Java 10**  | 2018         | Local-variable type inference (`var`), application class-data sharing, improved GC.                                                                        |
| **Java 11**  | 2018         | LTS release; new HTTP client API, `String` methods (repeat, strip, isBlank), running single-file programs without compiling, flight recorder open-sourced. |
| **Java 12**  | 2019         | Switch expressions (preview), Shenandoah GC, JVM constants API.                                                                                            |
| **Java 13**  | 2019         | Text blocks (preview), dynamic CDS archives, more GC improvements.                                                                                         |
| **Java 14**  | 2020         | Records (preview), pattern matching for `instanceof` (preview), helpful NullPointerExceptions.                                                             |
| **Java 15**  | 2020         | Sealed classes (preview), hidden classes, text blocks finalized.                                                                                           |
| **Java 16**  | 2021         | Records finalized, pattern matching `instanceof` finalized, JEP 338: Vector API (incubator).                                                               |
| **Java 17**  | 2021         | **LTS release**; sealed classes, enhanced pseudo-random generators, strong encapsulation of JDK internals, pattern matching for switch (preview).          |
| **Java 18**  | 2022         | Simple web server, UTF-8 by default, code snippets in documentation.                                                                                       |
| **Java 19**  | 2022         | Virtual threads (preview), structured concurrency (incubator), record patterns (preview).                                                                  |
| **Java 20**  | 2023         | Continuation of previews: pattern matching, virtual threads, foreign function & memory API (incubator).                                                    |
| **Java 21**  | 2023         | **LTS release**; virtual threads (Project Loom), record patterns, pattern matching for switch finalized, string templates (incubator).                     |
