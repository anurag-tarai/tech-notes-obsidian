# 1. DDL (Data Definition Language)

**Purpose:**  
Used to **define or modify database structure** such as tables, schemas, and indexes.

**Commands:**
- CREATE
- ALTER
- DROP
- TRUNCATE
- RENAME
- COMMENT

---
# 2. DML (Data Manipulation Language)
**Purpose:**  
Used to **insert, modify, and delete data inside tables**.

**Commands:**
- INSERT
- UPDATE
- DELETE    
- MERGE

---
# 3. DQL (Data Query Language)
**Purpose:**  
Used to **retrieve data from the database**.

**Commands:**
- SELECT
    
Common clauses used with SELECT:
- WHERE
- GROUP BY
- HAVING
- ORDER BY
- LIMIT
- OFFSET
    
---
# 4. DCL (Data Control Language)

**Purpose:**  
Used to **control access and permissions of users**.

**Commands:**
- GRANT
- REVOKE

---
# 5. TCL (Transaction Control Language)

**Purpose:**  
Used to **manage database transactions** and maintain data integrity.
**Commands:**
- COMMIT
- ROLLBACK
- SAVEPOINT
- SET TRANSACTION
- BEGIN / START TRANSACTION
