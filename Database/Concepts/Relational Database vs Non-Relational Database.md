|Feature|Relational Database (RDBMS)|Non-Relational Database (NoSQL)|
|---|---|---|
|Data Structure|Data stored in **tables (rows and columns)**|Data stored in **documents, key-value, graphs, or columns**|
|Schema|**Fixed schema** (structure must be defined)|**Flexible schema**|
|Relationships|Supports **relationships using keys**|Usually **no complex relationships**|
|Query Language|Uses **SQL**|Uses **NoSQL queries or APIs**|
|Data Consistency|Strong consistency|Often optimized for scalability|
|Best For|Structured data and transactions|Large scale, flexible, or unstructured data|
SQL Databases:
- MySQL
- PostgreSQL
- Oracle
- SQL Server

NoSQL Databases:
- MongoDB
- Cassandra
- Redis
- DynamoDB
