**Data** is **raw facts or values that can be processed to produce meaningful information.**
### Example
- **Data:**
```java
85, 90, 78, 92
```
    (just numbers)
    
- **Information:**
```java
Average marks = 86.25
```
    
    (meaningful result after processing)
---

### Example in DBMS
| Student_ID | Name   | Age | Course       |
| ---------- | ------ | --- | ------------ |
| 101        | Anurag | 23  | Java         |
| 102        | Rahul  | 22  | Python       |
| 103        | Priya  | 21  | Data Science |
Here:
- **23** is a piece of **data**.
- By itself, **23 has no meaning**.
- When combined with the column **Age**, it becomes **information**:  
    → _Anurag's age is 23._

### Key Idea

> **Data becomes information when it is processed and given context.**
