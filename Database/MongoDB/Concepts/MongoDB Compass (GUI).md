### What it is
- A **visual tool** to explore and manage MongoDB databases.
- Point-and-click interface with charts, filters, schema visualization.
### Best for
- Beginners.
- Visualizing data.
- Checking schema structure.
- Debugging queries.
### Pros
- Beginner-friendly, no commands needed.
- Great for exploring collections.
- Shows schema, indexes, performance.
- Easy to view documents & edit them manually.
### Cons
- Slower than shell.
- Cannot automate.
- Not suitable for scripting or production DevOps tasks.
- Heavier application.
### Example
To run a query, you simply type filter JSON:
```js
{ "age": { "$gt": 20 } }
```

And Compass shows the results visually.
