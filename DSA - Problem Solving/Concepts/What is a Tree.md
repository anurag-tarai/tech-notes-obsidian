> A **tree** is a non-linear data structure used to represent hierarchical relationships.
- It consists of nodes, where the topmost node is called the ****root****, and every other node can have one or more ****child nodes****.
- 
### Key Characteristics
- One **root** node
- Every node has **one parent** (except root)
- Can have **0 or more children**
- No cycles
### Real-life Examples
- File system
- Organization hierarchy
- HTML DOM
