**Amazon RDS (Relational Database Service)** is a managed cloud service by AWS that lets you easily set up, operate, and scale relational databases without handling servers, patching, or manual backups.

It supports multiple database engines:
- **MySQL**
- **PostgreSQL**
- **MariaDB**
- **Oracle**
- **SQL Server**
- **Amazon Aurora**

---

## ⚙️ Key Features

|Feature|Description|
|---|---|
|**Managed Service**|AWS automatically handles backups, patching, scaling, and replication.|
|**Multi-AZ Support**|High availability with standby replica across availability zones.|
|**Security**|VPC isolation, IAM authentication, SSL/TLS encryption, security groups.|
|**Scalability**|Change instance type or storage on the fly.|
|**Monitoring**|Integration with CloudWatch and enhanced monitoring.|
