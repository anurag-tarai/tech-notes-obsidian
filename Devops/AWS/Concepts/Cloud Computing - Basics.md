###### What is cloud Computing?
- Using computers, storage, and software over the internet instead of owning and managing them on your computer.
###### Cloud Services
---
1. IaaS -> `Infrastructure` as a Service
	- Provides **virtual servers, storage, networks** — you manage OS and apps yourself.
	- E.g - AWS EC2, Google Compute Engine, Azure VM
2. PaaS -> `Platform` as a Service
	- Provides **runtime environment, tools, databases** — you just deploy code.
	- E.g - Google App Engine, AWS Elastic Beanstalk, Heroku, Azure App Service, Render
3. SaaS -> `Software` as a Service
	- Fully managed **applications** accessed via web.
	- E.g -  Canvas, Gmail, Salesforce, Google Docs, Zoom
