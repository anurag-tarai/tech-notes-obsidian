### 1. Search the image 
```sh
docker search hello-world
```

### 2. Pull The image
```sh
docker pull hello-world
```

### 3. Create Container
```sh
docker create hello-world
```

### 4. Run the container
```sh
docker start <container_id>
```

### 5. Stop the container if running
```sh
docker pause <container_id>
```
