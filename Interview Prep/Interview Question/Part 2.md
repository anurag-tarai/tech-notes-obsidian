# Java + Spring Boot Interview Questions (Organized)

## 1. Introduction & Project Experience

These questions usually start the interview.

- Can you **introduce yourself and your recent project** in brief?
    
- Can you **explain your project flow** (sprint process, team workflow)?
    
- How do you **handle code reviews and feedback**?
    
- If you could **redesign one part of your last project**, what would you improve?
    

---

# 2. Software Design & OOP Principles

### Design Decisions

- If you have **Car and Engine classes**, would you use **inheritance or composition**?
    
- Have you **changed an existing design to support new requirements without breaking old code**?
    
- Which **principle explains modifying design without breaking existing code**?  
    (Open/Closed Principle)
    

### API Design

- How do you **design Java APIs so that other developers cannot misuse them**?
    

---

# 3. Java Object Model & Immutability

### Object Lifecycle

- Do you know the **object creation lifecycle in Java**?
    

### Mutable vs Immutable

- What are **mutable objects**?
    
- Have you faced **bugs due to mutable objects**?
    
- How can we **create immutable objects**?
    

### Advanced Immutability

- If an immutable class has a **List (ArrayList) field**, is it truly immutable?
    
- Why should we **not make fields public without getters/setters**?
    

---

# 4. Collections & Data Handling

### Choosing Collections

- How do you **decide the right collection for large data**?
    

### Equality Contract

- What is the **contract between `equals()` and `hashCode()`**?
    

### Concurrent Collection Updates

- How do you **safely update collections in multithreaded code**?
    

---

# 5. Concurrency & Multithreading

### Race Conditions

- Have you faced **race conditions**?
    
- How can we **avoid race conditions**?
    

### Synchronization

- What are the **limitations of the `synchronized` keyword**?
    

### Visibility & Atomicity

- What problems does the **`volatile` keyword solve**?
    

### Thread Management

- Do you know about **thread pools**?
    

---

# 6. Spring Boot & REST APIs

### Spring Boot Internals

- Explain the **lifecycle of an HTTP request in a Spring Boot application**.
    

### Dependency Injection

- How do you **manage dependencies using Spring Dependency Injection**?
    

### REST Best Practices

- What things should be handled for **validation in REST APIs**?
    
- How do you implement **REST API versioning**?
    

---

# 7. Database & Performance

### Query Performance

- A **query works fine locally but is slow in production**. How would you debug it?
    

### Indexing

- How do you **decide where indexes are required**?

# 8. Logging & Monitoring

- What **tools do you use for visualizing logs**?

# 9. Testing

- Suppose a program **calculates prime numbers in a range**.  
    How would you **write a JUnit test case** for it?

# 10. Microservices & Communication

- How do **two microservices communicate with each other**?
    

---

# 11. Git & Collaboration

- How do you **resolve conflicts when pushing code to GitHub or GitLab**?
    

---

# 12. Code Quality & Refactoring

- Can you tell me about a **Java code you refactored because it became hard to maintain**?
