**Spring MVC** is a **web framework within the Spring Framework** that follows the **Model-View-Controller (MVC) design pattern** to build **robust and maintainable web applications**.

### Core Components of Spring MVC
1️⃣ **Model**
- Represents the **data and business logic**
- Often implemented using **POJOs** or **service classes**

2️⃣ **View**
- Responsible for **rendering the user interface**
- Can be **JSP, Thymeleaf, HTML, JSON, or XML**

3️⃣ **Controller**
- Handles **HTTP requests**, processes input, calls services, and returns a response or view
- Annotated with `@Controller` or `@RestController`
