**ApplicationContext** is an **advanced Spring IoC container** that extends `BeanFactory` and provides **enterprise-level features** required for modern applications.

### Key Points

- Uses **eager initialization** by default - [Lazy & Eager Initialization](Lazy%20&%20Eager%20Initialization.md) 
    
- Supports annotations, AOP, events, and internationalization
    
- Automatically manages bean lifecycle
    
- **Default container used in Spring Boot**
