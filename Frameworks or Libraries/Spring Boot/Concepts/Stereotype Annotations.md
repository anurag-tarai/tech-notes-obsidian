**Stereotype annotations** are **specialized Spring annotations used to mark a class as a specific type of Spring bean**, helping Spring identify the role of the class in the application architecture.

## Main Stereotype Annotations in Spring

### 1️⃣ `@Component`

**Definition:**  
A **generic stereotype annotation** used to indicate that a class is a Spring-managed bean.

---

### 2️⃣ `@Controller`

**Definition:**  
Used to mark a class as a **web controller** that handles HTTP requests in Spring MVC applications.

---

### 3️⃣ `@Service`

**Definition:**  
Used to indicate that a class contains **business logic**.

---

### 4️⃣ `@Repository`

**Definition:**  
Used to mark a class as a **data access (DAO) component**, and it enables **exception translation**.
